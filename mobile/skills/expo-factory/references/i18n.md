# i18n — Coklu Dil Kurulumu

Orhan'in tum projeleri en az **TR + EN** destekli. Mobil uygulamada ayni kural.

## Stack

- **i18next** — core
- **react-i18next** — React bindings
- **expo-localization** — cihaz dili algilama
- Namespace sistemi — ekosistem ortak (common, auth, paywall) + proje-ozel (weather, livestock, vb.)

## Kurulum

```bash
bun add i18next react-i18next expo-localization
```

## Merkez Konfig

```typescript
// src/i18n/index.ts
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import * as Localization from 'expo-localization';
import AsyncStorage from '@react-native-async-storage/async-storage';

import trCommon from './locales/tr/common.json';
import enCommon from './locales/en/common.json';
import trAuth from './locales/tr/auth.json';
import enAuth from './locales/en/auth.json';
import trPaywall from './locales/tr/paywall.json';
import enPaywall from './locales/en/paywall.json';
import trWeather from './locales/tr/weather.json';
import enWeather from './locales/en/weather.json';

const STORAGE_KEY = 'user-language';

async function getInitialLanguage(): Promise<string> {
  const saved = await AsyncStorage.getItem(STORAGE_KEY);
  if (saved) return saved;
  
  const deviceLocale = Localization.getLocales()[0]?.languageCode ?? 'tr';
  return ['tr', 'en'].includes(deviceLocale) ? deviceLocale : 'tr';
}

export async function initI18n() {
  const lng = await getInitialLanguage();
  
  await i18n
    .use(initReactI18next)
    .init({
      resources: {
        tr: {
          common: trCommon,
          auth: trAuth,
          paywall: trPaywall,
          weather: trWeather,
        },
        en: {
          common: enCommon,
          auth: enAuth,
          paywall: enPaywall,
          weather: enWeather,
        },
      },
      lng,
      fallbackLng: 'tr',
      defaultNS: 'common',
      interpolation: { escapeValue: false },
      react: { useSuspense: false },
    });
}

export async function setLanguage(lng: 'tr' | 'en') {
  await AsyncStorage.setItem(STORAGE_KEY, lng);
  await i18n.changeLanguage(lng);
}
```

Root layout init:
```tsx
// app/_layout.tsx
import { useEffect, useState } from 'react';
import { initI18n } from '@/src/i18n';

export default function RootLayout() {
  const [ready, setReady] = useState(false);
  useEffect(() => {
    initI18n().then(() => setReady(true));
  }, []);
  
  if (!ready) return <SplashScreen />;
  return <Stack>...</Stack>;
}
```

## Kullanim

```tsx
import { useTranslation } from 'react-i18next';

function MyScreen() {
  const { t } = useTranslation('common');
  
  return (
    <View>
      <Text>{t('app.name')}</Text>
      <Text>{t('app.tagline')}</Text>
      <Button title={t('actions.continue')} />
    </View>
  );
}

// Namespace belirle
function Paywall() {
  const { t } = useTranslation('paywall');
  
  return (
    <View>
      <Text>{t('title', { appName: 'TarimIKlim' })}</Text>
      <Text>{t('plans.savePercent', { percent: 25 })}</Text>
    </View>
  );
}

// Birden fazla namespace
function Mixed() {
  const { t } = useTranslation(['common', 'paywall']);
  
  return (
    <View>
      <Text>{t('common:actions.save')}</Text>
      <Text>{t('paywall:title', { appName: 'TarimIKlim' })}</Text>
    </View>
  );
}
```

## Dil Degistirme UI

```tsx
// app/(tabs)/settings.tsx
import { useTranslation } from 'react-i18next';
import { setLanguage } from '@/src/i18n';

function LanguageSetting() {
  const { t, i18n } = useTranslation();
  
  return (
    <View>
      <Text>{t('common:settings.language')}</Text>
      
      <Pressable 
        onPress={() => setLanguage('tr')}
        style={{ 
          backgroundColor: i18n.language === 'tr' ? tokens.colors.primary : tokens.colors.surface,
        }}
      >
        <Text>Turkce</Text>
      </Pressable>
      
      <Pressable 
        onPress={() => setLanguage('en')}
        style={{
          backgroundColor: i18n.language === 'en' ? tokens.colors.primary : tokens.colors.surface,
        }}
      >
        <Text>English</Text>
      </Pressable>
    </View>
  );
}
```

## Namespace Stratejisi

**Ortak (ekosistem paylasimli):**
- `common` — actions, time, empty, errors (tum uygulamalarda)
- `auth` — login, signup, forgot password
- `paywall` — premium satis (her uygulamada ayni pattern)

**Projeye ozel:**
- `weather` (TarimIKlim)
- `livestock` (Hayvan Borsa Fiyatlari mobile)
- `nursery` (Bereket Fide mobile)

Ortak namespace'leri `@eco/shared-i18n` paketinde tutmak ideal — ama bu advanced, ilk iterasyonda her uygulamada kopyala.

## Sadelestirilmis Format Utilities

```typescript
// src/i18n/formatters.ts
import i18n from 'i18next';

export function formatCurrency(amount: number, currency = 'TRY'): string {
  return new Intl.NumberFormat(i18n.language, {
    style: 'currency',
    currency,
    minimumFractionDigits: 0,
    maximumFractionDigits: 2,
  }).format(amount);
}

export function formatNumber(num: number): string {
  return new Intl.NumberFormat(i18n.language).format(num);
}

export function formatDate(date: Date | string, options?: Intl.DateTimeFormatOptions): string {
  const d = typeof date === 'string' ? new Date(date) : date;
  return new Intl.DateTimeFormat(i18n.language, {
    year: 'numeric',
    month: 'long',
    day: 'numeric',
    ...options,
  }).format(d);
}

export function formatTime(date: Date | string): string {
  const d = typeof date === 'string' ? new Date(date) : date;
  return new Intl.DateTimeFormat(i18n.language, {
    hour: '2-digit',
    minute: '2-digit',
  }).format(d);
}
```

## RTL Destek (Opsiyonel — Arapca, Farsca)

```tsx
import { I18nManager } from 'react-native';
import * as Updates from 'expo-updates';

const rtlLanguages = ['ar', 'he', 'fa'];

async function setRTLIfNeeded(lang: string) {
  const shouldBeRTL = rtlLanguages.includes(lang);
  if (I18nManager.isRTL !== shouldBeRTL) {
    I18nManager.forceRTL(shouldBeRTL);
    await Updates.reloadAsync();  // app'i yeniden baslat
  }
}
```

Ekosistem icin su an TR + EN yeterli, RTL ileride.

## Yapma Listesi

- Hard-coded string **YASAK** — her tekst i18n'den gelir
- Locale dosyasinda JSON syntax error — CI'da validate et (`jsonlint`)
- Ingilizce string'i varsayilan olarak Turkceyle karistirma — namespace disiplinle
- `console.log(t('key'))` ile debug yapmadan once `useTranslation` hook'un kuruldugunu kontrol et
- Performansta: ayni ekranda ayni namespace'i tekrar tekrar import etme, tek bir `useTranslation` cagrisi yeterli
